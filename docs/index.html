ESP-NOW Mesh

Connectionless 2.4 GHz protocol. Nodes communicate directly — no Wi-Fi AP required between sensor and coordinator.

🔒
CRC-8 Validated

Every packet is validated with CRC-8/MAXIM polynomial 0x31 over all 31 header + payload bytes before acceptance.

0x31 MAXIM
🔁
Relay Forwarding

Relay nodes extend coverage. Deduplication cache prevents re-broadcast storms. Hop counter enforced at MAX_HOPS = 4.

DEDUP_CACHE = 16
🌡️
Multi-Sensor

DHT22 for temperature and humidity. MQ-135 for CO₂ estimation and VOC raw ADC. AQI tier computed on-device.

Coordinator Aggregator

// wire protocol

32-byte fixed packet.

struct __attribute__((packed)) — no alignment padding, no surprises.

Byte   0      1      2      3      4      5      6      7
      ┌──────┬──────┬──────┬──────┬──────┬──────┬──────┬──────┐
      │ VER  │ TYPE │ SRC  │ DST  │ SEQ  │ HOP  │FLAGS │ LEN  │
      └──────┴──────┴──────┴──────┴──────┴──────┴──────┴──────┘

Byte   8 ──────────────────────────────────────────────── 30
      ┌────────────────────────────────────────────────────────┐
      │              PAYLOAD  (23 bytes — SensorPayload)      │
      ├──────────┬─────────┬──────┬──────┬──────┬─────┬───────┤
      │ int16   │ uint16  │uint16│uint16│uint16│uint8│uint32 │
      │ temp×100│ hum×100 │ pm25 │ co2  │ voc  │ aqi │ ts_ms │
      └──────────┴─────────┴──────┴──────┴──────┴─────┴───────┘

Byte   31
      ┌──────┐
      │ CRC8 │  ← CRC-8/MAXIM (poly 0x31) over bytes 0–30
      └──────┘

TYPE:   0x01 DATA   0x02 ACK   0x03 HEARTBEAT
FLAGS:  bit0 = ACK_REQ  │  bit1 = RELAYED  │  bit2 = ENCRYPTED (future)
AQI:    0=GOOD  1=MOD  2=USG  3=UNHEALTHY  4=V-UNHEALTHY  5=HAZARDOUS

// setup

From zero to packet received.

Five steps. Don't skip the MQ-135 warm-up.

1

Flash mac_scanner on every ESP32

Open Serial Monitor at 115200 baud. Copy the array-format output for each board — coordinator, sensor nodes, relay.

{ 0x24, 0x6F, 0x28, 0xAB, 0xCD, 0xEF } // paste into config.h
2

Configure sensor_node/config.h

Set a unique NODE_ID for each sensor node, paste the coordinator MAC.

#define NODE_ID 0x01 uint8_t COORDINATOR_MAC[6] = { 0x24, 0x6F, ... };
3

Set credentials locally (never committed)

Copy the example file and fill in your network credentials. This file is gitignored.

cp coordinator_node/config_local.h.example \ coordinator_node/config_local.h # then edit: WIFI_SSID, WIFI_PASSWORD, DASHBOARD_URL
4

MQ-135 calibration

Power on sensor node outdoors. Wait 30 minutes. Read the voc_raw value from Serial. Set MQ135_CLEAN_AIR to that value in config.h.

5

Flash in order: Coordinator → Sensor → Relay

Coordinator must be listening first. Confirm success on coordinator Serial Monitor.

[RX] Node:0x01 SEQ:000 Hop:0 RSSI:-52dBm CRC:OK [SENSOR] Temp:27.54°C Hum:65.23% CO2:612ppm AQI:1
